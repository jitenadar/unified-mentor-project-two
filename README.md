Little Steps — 24x7 Childcare Platform
India's trusted platform connecting parents with verified creches, daycare centres, and babysitters — available day, night, and emergency.

Live Site: https://littlesteps-5c0.pages.dev/

About
Little Steps is my second internship project at Unified Mentor. It is a childcare marketplace for Indian parents who need flexible, round-the-clock childcare — especially night-shift workers and single parents.

Features

Smart filters — age group, timing, plan type
Police-verified and background-checked providers
24x7 availability — even at 2 AM
Instant booking — Hourly, Daily, Monthly plans
Provider dashboard for managing bookings
Covers 15+ cities across India


Built With
HTML5 — structure, layout, and styling all written in a single file, deployed on Cloudflare Pages.
